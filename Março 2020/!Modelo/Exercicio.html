Banco de palavras

5.6.5Montar um módulo com instruções para manter um banco de palavras. Deve haver instruções para incluir ,alterar e excluir uma palavra do banco, além de pesquisar se uma dada palavra está no banco. O banco é armazenado em um arquivo em disco(ou no CACHE do navegador). Todas as instruções devem ser reutilizáveis.
